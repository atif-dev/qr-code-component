Requirements
--------------------
1)Get it looking as close to the design as possible.
2)You can use any tools you like to help you complete the challenge. So if you've got something you'd like to practice,
  feel free to give it a goal.
3)Download the starter code and go through the README.md file. This will provide further details about the project.
  The style-guide.md file is where you'll find colors, fonts, etc.


---Getting started

    Set up the project with version control (e.g. Git)

    Read the README.md file and have a look around the project

    Get colors, fonts etc from the style-guide.md file

    Set up your project/file architecture however you want

    Start coding!
